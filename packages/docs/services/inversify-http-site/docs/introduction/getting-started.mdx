# Getting started

Start by installing `inversify` and `reflect-metadata`:

```bash
npm install inversify@alpha reflect-metadata
```

:::warning

Make sure to enable [Experimental decorators](https://www.typescriptlang.org/tsconfig/#experimentalDecorators) and [Emit Decorator Metadata](https://www.typescriptlang.org/tsconfig/#emitDecoratorMetadata) options in your `tsconfig.json` to use this library.

:::
